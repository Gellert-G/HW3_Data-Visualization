# Za'atari Population Explorer

This folder is a standalone D3 project for task 2.

## Files

- `index.html` - page entry
- `styles.css` - custom page styling
- `app.js` - D3 chart, brush, tooltip, and statistics
- `data/zaatari-refugee-camp-population.csv` - population dataset

## Open

Double-click `index.html` to open it in a browser. The page only needs internet access for the D3 CDN.
