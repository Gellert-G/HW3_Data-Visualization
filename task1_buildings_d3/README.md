# Top Completed Skyscrapers

This folder is a standalone D3 project for task 1.

## Files

- `index.html` - page entry
- `styles.css` - custom page styling
- `app.js` - D3 interaction and rendering
- `data/buildings.csv` - building dataset
- `img/*.jpg` - local building photos

## Open

Double-click `index.html` to open it in a browser. The page only needs internet access for the D3 CDN.
